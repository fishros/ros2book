2.5 ROS 2 Basics: Programming

Compared to ROS 1, ROS 2 adopts newer versions and modern features in its development. For example, the current Humble version of ROS 2 uses Python 3.10 and leverages new features from C++ versions 11, 14, and 17. “A tall building rises from the ground.” Before diving into these new features, let’s first learn about one of the most important characteristics of modern high-level programming languages: object-oriented programming (OOP).
