Introduction
=================================

Through the learning in Chapter 1, you have mastered how to install ROS 2 and how to start a ROS 2 program using the command line. However, being satisfied with just this is clearly not enough, as mastering the installation and launching of ROS 2 programs is only half the journey of getting started. It’s time to learn something new. Today, we’ll learn where to write ROS 2 programs and how to compile code to generate executable files.
