3.5 ROS 2 Basics: Introduction to Git
=====================================

In addition to the system itself, another fascinating aspect of ROS 2 is its powerful community. On code hosting platforms such as GitHub, there are countless excellent open-source programs available for us to use. These programs without exception are managed with Git for version control. Therefore, mastering Git is essential to better embrace the ROS 2 ecosystem.

Git is a free and open-source distributed version control system tool. It was developed by Linus Torvalds, a Linux kernel developer, in 2005. In the previous learning sessions, we have already installed Git. Now, let's dive directly into learning how to use it.
